# 📘 Лабораторная работа №1  
## Работа со словарями (часть 1 из 3)

---

## 📌 Описание

Данная лабораторная работа является **первой из трёх**, посвящённых разработке системы работы со словарями.

В рамках этой работы реализовано простое консольное приложение на Java, позволяющее работать со словарями, хранящимися в файлах на диске.

---

## 🎯 Цель работы

Разработать библиотеку (сервис) для работы со словарём и консольное приложение для взаимодействия с пользователем.

---

## ⚙️ Функциональность

### 📚 Работа со словарём

Реализованы следующие операции:

- чтение словаря из файла
- просмотр всех записей
- поиск записи по ключу
- добавление новой записи
- удаление записи по ключу

---

### 🌐 Поддержка двух словарей

Приложение работает с двумя словарями:

#### 1️⃣ Словарь №1
- ключ: слово длиной **4 символа**
- допустимые символы: **латинские буквы (a-z, A-Z)**

#### 2️⃣ Словарь №2
- ключ: слово длиной **5 символов**
- допустимые символы: **только цифры (0-9)**

Значение в обоих словарях — перевод слова на русский язык.

---

## 🖥️ Консольное приложение

Пользователь может:

- просматривать содержимое обоих словарей
- выбрать словарь для работы
- выполнять операции:
  - добавление записи
  - поиск по ключу
  - удаление записи
